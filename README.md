API, json server, axios kullanarak task projesi yapıldı.

npm create-react-app
<br/>
npm install -g json-server
<br/>
npm install axios
<br/>
json-server --watch db.json --port 3004
<br/>
npm start
<br/>

kodları terminale girildikten sonra proje çalıştırılabilir.
